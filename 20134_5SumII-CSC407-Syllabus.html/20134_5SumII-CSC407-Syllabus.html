  • Objective of class

    1. To pick up from where CSC 373/406 left off: looking at computing systems at grander scales than the CPU
    2. To walk you thru the arc the C/C++ programs traverse: 
      source code -> object code -> executable code
    3. To give you, as a programmer, a idea of the services that the OS provides to help you write programs
